# Projet : Jeu "Snake"
    Zapharaos, Matthieu Freitag

Voici le jeu "Snake" réalisé en JAVA.

### Le but du jeu :
____________

- L'objectif est de manger le plus de pommes possible. Vous pouvez voir votre score en haut à droite de la fenêtre, votre record au centre et le chronomètre en haut à gauche.

- Attention : si vous percutez les bords ou un morceau de votre queue, c'est perdu.

### Début du jeu :
____________

- Pour débuter la partie, cliquez sur "START"

### Pour se déplacer :
____________

- Avec les flèches directionnelles.
- Avec les touches "Z", "Q", "S", "D"."

### Faire pause :
____________

- Vous pouvez faire pause avec la touche "ESPACE" ou "ECHAP".
- Pour reprendre la partie, appuyez sur le bouton "RESUME" ou les touches "ESPACE" ou "ECHAP".

### Pour recommencer une partie :
____________

- Quand vous avez perdu, vous pouvez recommencer une partie avec le bouton "RESTART" ou la touche "R".

### Pour quitter le jeu :
____________

- Vous pouvez simplement fermer la fenêtre ou cliquer sur le bouton "QUIT GAME" quand vous perdez ou faites pause.
